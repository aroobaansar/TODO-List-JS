# To-Do List App (JavaScript)

A simple and interactive To-Do List web application that allows users to add, complete, and delete tasks with persistent storage using localStorage.

##  Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Tasks saved in browser localStorage
- Automatic task loading on page refresh

##  Technologies Used
- HTML
- CSS
- JavaScript (DOM manipulation & localStorage)
